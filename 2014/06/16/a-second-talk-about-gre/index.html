再谈GRE考试

Posted on 2014-06-16 Edited on 2017-07-08 In tutorial Disqus:

本人应邀写下了第二篇，应该也是最后一篇关于GRE的文章了，希望能为大家提供帮助。

为什么是第二篇呢？因为之前曾经写过一篇关于备考GRE的文章，大体内容是对备考过程的回忆和一二战的反思，可惜现在拿过来似乎已经过时了。经历了一整年的磨难闯过了这个关口，回过头看看，有不少新的体会，在此简要叙述一番。

这次主要谈谈作文。作文是我一直以来的硬伤，本来去年六一的成绩已经基本够用，但考虑到那个作文成绩实在太难看，于是才有了二战的惨败，以及三战的反击。

总结一下之前考试的教训，作文的问题主要集中在这几个方面：

  • 临场发挥没有思路，不知道写什么
  • 打字太慢，有想法也写不出来
  • 写出来的东西很难看，不符合阅卷人的口味
  • 写一半跑题了

下面讲解如何对症下药逐个击破。


本人以为对argument而言不存在思路问题，熟练的学生看一遍题应该熟练到可以直接找出3到4处错误，如果做不到就只能多看题多思考了。而issue嘛，我也必须承认，对于某些艺术类啊政治类的题目确实不知如何下手。为了避免考场上因被吓傻而出现脑抽手抖的情况，平时应作两手准备，一是到网上看看最近出现率比较高的题目，有针对性练习一下，二是买本《GRE写作5.5》看一看对所有题目的分析，确保心中有数。当然必须指出的是，题目看一遍就好，重要的不是对题目本身有多熟悉，而是看到题目之后能够多熟悉地写出来。抽象来说，输入是不能代替输出的，考官只看输出，也就是你写的文章，而从来不管输入，也就是你怎么准备材料的，所以说输入是为输出服务的。如果可以在有限的时间写得一篇好文章，现场读题也完全可行。

除此之外，还有一个解决写作没思路的好方法，就是写例子，而且越长越好。我经常会发现一些抽象的句子似乎很容易用中文表达出来，但若要转换成英文，就会抓耳挠腮不知所措。不过，对表述具体事实的句子，用英文写则会感觉舒服得多。所以不要为第一段如何充分准确地表达观点而煞费苦心了，用一句话把大致的意思讲清楚（赞成、反对、均衡观点），剩下的时间开始做分析码案例证明观点即可。

相较之下，打字慢是个很难解决的问题。要知道，在考官看来字数太短的文章是没有说服力的，尤其是机阅环节，字数是一个重要的拉分项，因此确保文章的长度就成为了提高分数的第一步。一篇文章限时30分钟，也就是1800秒，刨除看题和检查花掉的200秒，在剩下的1600秒之内写出一篇400词的文章，意味着每4秒钟就要输入一个单词。这个速度当然不能算很快，但考虑到打字间隙的思考时间，以及经常性地哪个地方打错了需要按后退键，或是人品不好遇到哪个词忘记了拼写需要试着打几遍，整个过程还是相当紧张的，或者说，双手几乎没有空闲的时候。这个时候，如果思路通畅，打字速度快的人自然就占了优势。那些ETS官方提供的5分、6分范文，动辄上了600词，别说思路了，就是手速咱们也达不到啊！这个时候就需要做一些针对性的练习了。

首先推荐一个不靠谱的方案，就是买一个机械键盘Orz。估计大多数人都是用自己的笔记本做写作训练，但是不得不提的是笔记本键盘的手感和考场上键盘的手感有明显的差别。键程、触发力度、反馈力度、声音，这些差异可能会给考场的打字速度带来一些影响。为了更加熟悉考场上键盘的感觉，在此推荐大家购买一个类似的键盘，当然机械键盘就更好了。嗯不过这只是个建议而已，因为那个东西有点儿贵而且也不是很必要，但是不得不说手感和声音都极其好（我说的是青轴），就算不能带来明显的手速提升，可能也会让你喜欢上打字和写作。当然鉴于该设备笨重，且噪音比较大，请勿在自习室和图书馆使用。等到在类似的键盘上可以轻松盲打出"The quick brown fox jumps over the lazy dog"，就可以自称对考场的键盘足够熟练了。

下面说一个正经的解决办法，就是做copy练习。装一个比较性感的文本编辑器，比如Sublime Text，事先找一些英文文章，左右分屏，边看边打，强迫自己不看键盘，以及几乎不用后退键。本人做过一段时间的练习，感觉效果还不错。后来因为搞开发学了vim，在vim里面分屏打字是一件很酷的事情。不过后遗症就是vim打了错字总习惯用Esc-d-b三个按键删了重打，但是考场的编辑器没有这么高端的功能，所以考试前几天又换回了Sublime Text。有兴趣的同学可以试用一下。


关于作文的评判标准，只看官方的OG是不够的（或者说太抽象），最好的办法是找些范文参考。官方的范文我都通读过一遍，然而发现了一个很沉痛的事实————这文章绝壁不是中国人写的，而且正常的中国人也写不出这种文章（尤其issue）。在这种情况下，琢磨他们是怎么写的没有明显的意义，我摸索出来的解决方案是看中国人自己写的范文。在此强烈推荐极智批改网提供的一些范文（此处不代表我推荐此网站……）。我对这些文章做了进一步的整理，发布到了GitHub上，方便大家查阅和下载。经过我的研究，这些文章大多具有如下套路：

  • Argument大多是结论-证据-反驳的开头；正文段一般有4段，一个逻辑漏洞视情节严重与否用一段或两端论述；结尾段大多是结论+进一步支持的论据。
  • Issue均衡观点和一边倒都有出现，正文段可以只有两端。

下面结合本人的经验详细论述一下。


Argument的结构几乎完全固定，可以发挥的余地几乎仅限于举出几个合乎情理的例子说明原文的缺陷。一般而言4个正文段的文章如果字数不足420~450，是不容易把每一段阐释清楚的，因此对手速比较慢的童鞋我强烈建议正文只写三段，不过一段可以包含一个漏洞多个方面的问题。对每一处缺陷，不仅要指出缺陷的类型，而且需要给出一个例子来说明这种我们假设的情况如何会发生。例如调查可能缺乏代表性的问题，就要说可能抽样不均，一部分人被抽的比例远比另一部分人大；再如全国的情况不适用于当地，就要说地方的经济实力和全国不是一个水平，或者地方的人口素质更高更守规矩，或者人口结构不同没有可比性；再如观察值不一定代表实际情况的问题，就要说警察可能有指标啊因此会选择瞒报，或者观察鹿群的猎人自己有偷猎行为。总之一个具体的情景会比一个抽象的名词有意义得多。

关于issue的两种观点————一边倒战略和均衡观点战略，我个人觉得均可。《GRE写作5.5》是强烈推荐均衡观点和让步观点的。这么做优势很明显，就是正反两个方向都要说，因此很容易把文章写长，不过万一打字不够快或者说明不够充分也是很容易Orz的。我考前练习的时候几乎全是一边倒的写法，正文只有两段，每段分析+一个例子，简单递进关系，然后就得出结论了，写过3~5篇之后可以稳定在350词左右。考试那天的题目是“所有大学生都可以从国外一个学期以上的交换中获益”，我打算反驳这个观点但是理由不太充分，最后就写成了均衡观点，结果似乎还算不错。

现在回想起来，把issue写好的一个很重要的方法也是写具体生动的例子，既可以把文章写长又可以言之有物。举考场上那篇文章的例子，为了说明交换是有好处的，我就说现在讲求国际合作，一个未来的雇员（也就是大学生）最好能够和其他国家的人协同工作，而交换是培养这个能力的最好方法之一。比如google和amazon这些跨国公司就看重交换的经历，所以交换过的学生就有优势，这样就切合题目的benefits了。虽然事实不一定就是如此，但只要逻辑是对的，文章就是有说服力的。当然，考场上还是不要举什么奶茶或者力宏之类的例子了。有了前文的让步后文我开始大谈特谈交换的不值，比如说现在有coursera有udacity有edX，如果单纯从学术的角度出发，国外牛b的课程咱们也都能上，而且还是免费的，费不着为了几门课花这么多钱；另外咱们不是有夏令营嘛，学习与老外相处不一定需要一个学期以上的时间啊；还有现在咱们不是有skype和虚拟现实技术嘛，都不要亲自去了宅在家里一样走遍世界！（这么想其实是不好的不过考场上言之有理即可，考官从来不看三观）最后就是办手续麻烦，如果人品不好出了什么事情，人生地不熟就只能呵呵了。总之这么几条下来文章就比较充实了。


最后谈一下如何应对跑题的情况。先前我一直在说不要太重视立意，只要把握住情感倾向就行了。毕竟立意这个东西很虚，咱们不容易讲清楚。但是有时候会遇到在正文段写例子很顺，写到很high的时候，忽然发现例子跑偏，无法有效支持论点，这是让人很不爽的事情，而且这种事情我还遇到过好几次。这个时候可以先不管这么多，感觉能用的例子就先写上，然后每写完一段就停顿一下，看看这个例子是否切合观点。如果觉得有出入，此时有两种做法，一是修改例子的表述方式，二是修改论点。从实际的可操作性来看第二种做法优势明显，因为论点往往是第一段的倒数第一句或者倒数第二句话，很容易定位，而且想要表达一个不同的观点有时候只要替换掉几个词就好了。我在做练习的时候就有过多次类似的经历，本来是写一边倒的文章，写了一半发现例子本身有漏洞，不能有效地支持观点，这时候干脆选择下一段把这个漏洞指出来，改成平衡观点的文章。总之在考场上时间紧迫，不容得那么多思考，想到什么就写什么，学会随机应变是很重要的。

我前两次考试都没练过作文，结果都是一败涂地，所以三战的时候痛下决心，一天练了8篇限时作文（3 issue 5 argument），效果拔群。所以说考试什么的，一方面我们要承认功夫在平时，但是另外一方面也必须重视考前的突击，到了最后一天也不要放弃治疗。

顺带提及一下语文和数学吧。

有人推荐了magoosh测评系统，本人测试了一下，认为难度非常符合160+的verbal真题水平，尤其是在填空部分：生词很多，选项看不懂；就算选项看懂了，句子意思也看不懂；两者都看懂了也选不对。考前最后一周我拿这个刷填空，正确率一直在10%左右徘徊，基本上就当背单词了。在此推荐喜欢飙高分的同学使用。如果不想到官网上花钱买题，淘宝上或者论坛里有下载的链接，可以花时间找一下。

再多说一句，真正考试的时候填空还是比较简单的，如果句子长则单词一般都看得懂，如果单词看不懂则句子应该很很好理解，总之150+水平的题目用《再要你命3000》应对问题不大。

阅读继续推荐36套，因为找不到更好的练习材料了。练阅读我的习惯是从不计时，几乎要把每句话都看懂才开始做题，这样有助于提升理解能力。36套刷了不到一半，正确率从60%上升到了80%多。在考场上可能会感觉时间略有紧迫，不过只要填空不浪费太多时间阅读是可以做完的。

关于数学，考得只有细心二字。三战我做完每个数学部分之后都没有直接点继续进入下一部分，而是反复检查，终于避免了阴沟翻船的惨剧。对本次考试的阴沟记忆犹新：

x和y是整数，-7 <= x <= 3， -2 <= y <= 4，求x*y的最大值与13的大小关系。

答案是大于，我第一遍选的答案竟然是小于，这(>_<)

最后祝大家考试顺利！
