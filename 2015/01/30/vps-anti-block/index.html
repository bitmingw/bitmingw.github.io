VPN 连不上怎么办？

Posted on 2015-01-30 Edited on 2023-12-16 In tutorial Disqus:

最近一段时间政府加强了对互联网的管控，让不少用 VPN 刷 instagram 的童鞋不开心了。你们的 VPN 估计一时半会儿还连不上，我也不能帮你们上 instagram, 不过我倒是可以让你们在电脑上收 Gmail 刷 Facebook 看 YouTube. 下面隆重推出翻墙神器 VPS!


VPS 是神马？

大家应该一眼就看出 VPS 和 VPN 的缩写非常相近。VPS 即“虚拟专用服务器”，它也是通过转发网络请求的方式实现翻墙的。不过与 VPN 不同，它并不使用 PPTP, L2TP 等专用的网络协议，而是使用 SSH 这种通用的协议，因此不容易被封杀。


如何使用 VPS？

首先给大家道个歉，VPS 暂不支持手机和平板设备，只有 PC 机可以使用。

【第一步：注册账号】 我没有时间写一个华丽的页面，如果你需要 VPS 服务，短信微信QQ联系我，或者给我发邮件 bitmingw[at]gmail[dot]com, 我会帮你完成注册。

【第二步：下载客户端】 客户端有多种选择，我推荐使用 PuTTY. 它是绿色软件，只有一个 exe 文件。

【第三步：配置客户端】 这里需要稍微花一点儿时间。在 Session 选项卡中，把我提供的主机地址填上（图中的地址仅为示意）。

在 Connection 下的 Tunnels 选项卡中，如图填写 Source port 和 Destination，再点击旁边的 Add 按钮即可。

为了避免每一次使用都重复设置，可以回到先前的页面，在 Saved Sessions 一栏下输入一个名字，点击 Save 以保存设置。下次使用时只需选择相应的设置，再点击 Load 即可。

【第四步：连接服务器】 设置完毕后，点击右下角的 Open 按钮，先后输入用户名和密码就可以登录。你看到的界面可能与我的稍有不同。

【第五步：更改软件代理设置】 最后需要修改浏览器或者应用软件的设置，让他们通过服务器转发网络请求。以 IE 为例，在浏览器或控制面板中打开 Internet 选项，点击右下角的 LAN 设置。

开启代理服务器，并按下图设定地址和端口号即可。

Chrome 浏览器默认使用 IE 的代理设置，如果你安装了 Proxy SwitchySharp 插件，参照 IE 的方法配置即可。

此外，许多软件都支持代理服务器转发，例如 Dropbox.

【第六步：下线与恢复】 停止使用网络时，请与服务器端开连接。点击 PuTTY 右上角的关闭按钮即可。为了避免下次使用出现问题，请恢复上一步做出的设置，停用代理服务器功能。


FAQ

  • VPS 可以看 YouTube 么？

可以看。不过用得人多了之后则未必。由于服务器性能和带宽有限，不建议使用视频等大流量服务。

  • 本服务免费？

没错，搭建 VPS 是我的个人行为，不会收取任何费用。

  • 安全和隐私方面有什么保证吗？

服务器会暂时记录您访问的地址，但不会记录其他任何内容。通过服务器的转发，可以在一定程度上隐匿您的公网 IP 地址。

  • 我可以用服务器的资源做一些其他的事情吗？

理论上是可以的，比如说在上面存储一些数据。但我对您存储的数据不作任何保证。

运维FAQ

系统运行几天来，收到了多位用户的反馈，现在就普遍遇到的问题进行解答。

  • 账户无法登录，输入用户名之后无法输入密码

这是因为服务器使用了 Linux 系统，在用户输入密码的时候不会显示任何信息。系统依旧会记录您的登录信息，输入正确的密码按下回车后就可以正常登录了。

  • 连接速度慢，连接五分钟之后自动掉线

与服务器的连接是否稳定，以及最终的网络访问速度，与用户所在地的网络状况有直接的关系。如果用户本身的网络不佳，则很难获得稳定的服务体验。

  • 刚刚连接系统就会报错，客户端进入 inactive 状态

这是因为在客户端与远程服务器的连接尚未完全建立时，就通过客户端的端口转发功能向服务器发送代理请求，这样的行为会触发客户端的网络异常。解决的办法是，先让客户端连接至服务器，待连接稳定后再设置相关软件的代理服务器选项，这样就不会因过早发送请求而触发异常了。如果客户端进入了 inactive 状态，请重启客户端软件，取消代理服务器选项，并重新连接至服务器。
